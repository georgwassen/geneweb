GeneWeb: mode d'emploi


La présente documentation fait parfois référence à des fenêtres de commandes interactives. Il existe désormais un programme "gwsetup", plus simple d'utilisation, permettant d'éviter d'avoir à les utiliser.

    • Comment démarrer avec GeneWeb.

    • Les restrictions d'accès à la base de données.
    • Comment faire des modifications en ligne dans la base de données.
    • Comment fusionner des bases de données.
    • Nettoyer, renommer, sauvegarder des bases de données: maintenance.

  • Vous voulez participer à l'amélioration de GeneWeb:

    • Comment faire des remarques, suggestions, rapports d'anomalie, contributions.

La page d'accueil de GeneWeb est la suivante.

  • http://cristal.inria.fr/~ddr/GeneWeb/
